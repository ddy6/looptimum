# SnappyHexMesh Campaign06

Sanitized case-study package for a bounded Looptimum mesh-control campaign.

Contents:

- `case_study.md`: public narrative and key takeaways
- `campaign/`: archived optimization-state artifacts
- `validation/`: sanitized derived validation summaries and plots for the
  selected coarse candidate

The public package is intentionally sanitized:

- case names are genericized
- branch-level validation metrics are anonymized
- raw solver artifacts are not mirrored directly
