# Toy Objectives (Underscore Path)

This directory holds dedicated end-to-end toy loop examples used for quick
integration confidence checks.

Included:

- `03_tiny_quadratic_loop/`: deterministic noisy-quadratic loop with
  `suggest -> evaluate -> ingest -> status` flow

Note:

- Existing integration-pattern examples remain under `examples/toy-objectives/`
  (hyphen path). Both directories are intentional during this transition.
